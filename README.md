# 🛏 床墊傢俱購物車專案（前台 + 後台）

這是一個使用 純 JavaScript + HTML + CSS 完成的床墊傢俱購物車作業，  
包含前台購物流程與後台訂單管理功能，並搭配 Axios 與 C3.js 進行資料呈現與互動。

作品展示了 **完整 CRUD、表單驗證、購物車邏輯與後台資料分析能力**。

---

## 🔗 Demo

👉 前台購物車頁面：https://scarlett-terra.github.io/JS_Project_20251122/apiEight.html    
👉 後台管理頁面：https://scarlett-terra.github.io/JS_Project_20251122/admin.html 

---

## 🚀 使用技術

- HTML5 / CSS3 / JavaScript (ES6)  
- Axios (前後台資料串接)  
- C3.js / D3.js (後台圖表展示)  
- SweetAlert2 (彈出提示)  
- 金額格式處理（千分位）  
- DOM 操作與事件綁定  

---

## 📌 前台功能 (Frontend Features)

1. 商品展示與篩選  
   - 使用 Axios 顯示產品卡片  
   - 可依類別篩選商品  

2. 購物車操作  
   - 新增商品至購物車  
   - 購物車數量增減  
   - 刪除單一商品或清空購物車  

3. 訂單送出與表單驗證  
   - 客戶資訊驗證（未填寫提示紅字）  
   - 彈出提示提醒資料未填寫完成  
   - 訂單成功送出提示 (SweetAlert2)  
   - 金額千分位顯示  

---

## 📌 後台管理功能 (Backend Dashboard Features)

1. 圖表分析  
   - 顯示前三名產品排行榜  
   - 超過第四名統整為「其他」  

2. 客戶訂單管理  
   - 顯示客戶訂單列表（日期由最新至最舊）  
   - 總購買內容展示  
   - 已處理 / 未處理狀態調整顯示顏色  
   - 刪除單一客戶或全部客戶

## ✅ TodoList 介面

這是一個使用 **純 JavaScript + HTML + CSS** 製作的待辦清單專案，  
可以記錄、管理日常待辦事項，並實現簡單的增刪改查功能。

### 🔗 Demo
👉 TodoList 頁面：https://scarlett-terra.github.io/JS_Project_20251122/toDoList.html

### 🚀 使用技術
HTML5 / CSS3 / JavaScript (ES6)  
DOM 操作與事件綁定  
LocalStorage（資料永久儲存）  
事件監聽（新增、刪除、勾選待辦）  
篩選功能（全部 / 待完成 / 已完成）  

### 📌 功能特色
- 新增待辦事項  
- 刪除單一待辦或清空已完成項目  
- 勾選待辦事項（標記已完成 / 未完成）  
- 篩選待辦清單（全部、待完成、已完成）  
- 本地永久儲存（刷新頁面不會遺失資料）
